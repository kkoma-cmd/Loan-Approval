## Loan Approval Program.
	
	This program checks bank approval based on the user's: 
		- Annual Income 
		- Number of years at the company
	
	The user enters values using JOptionPane
					----	
## Requirements
	
	Java JDK (17+)
	Eclipse IDE
					----	
## How to run this program?
	
	I. Open the project in eclipse.
	II. Run Loan.java.
	III. Enter your  Annual Income and Number of years at the company.
